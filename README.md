# Doodle Jump (itch.io Core Mechanics Challenge Jam, Spring 2026)

![screenshot of the game](docs/images/thumbnail.png)

Here is my entry for the [itch.io Core Mechanics Challenge Jam, Spring 2026](https://itch.io/jam/core-mechanics-challenge-2026-spring). The task/theme for this jam is to make a game with the same core mechanic as "[Doodle Jump](https://en.wikipedia.org/wiki/Doodle_Jump)." Here is the task description from the jam page:

> ## TASK (Theme): Doodle Jump
> 
> https://en.wikipedia.org/wiki/Doodle_Jump
>
> **Core mechanic**: Endless one-way platform jumping
> 
> Rules:
> 
> - Player continuously moves one way by jumping between platforms
> - Platforms are spawned vertically with increasing challenge
> - Some platforms may move, break, or disappear
> - Game ends when the player falls off the screen
> 
> Optional features: (unlocked during the jam)

## My Submission

**Title:** Kamil's Climb

**Description:** You play as Kamil, a cute Rocky Mountain bighorn sheep on a mission to climb as high as he can up a waterfall — until he inevitably tumbles back down (don't worry, he always splashes down safely). Jump between platforms as they get harder to reach, and see how high you can go before the fall.

**Submitted:** May 5th 2026

**Platform:** itch.io (web build)

**What actually shipped:**

This was my first game jam. I originally planned an ambitious card-collecting mechanic where landing on platforms would build poker hands and trigger boost jumps. I deliberately spent a lot of time planning upfront (see ROADMAP.md), but about halfway through I realized I wouldn't have time to complete the card mechanic. I pivoted to delivering a minimal but complete Doodle Jump-style game instead.
